# Pig Game

A browser-based implementation of the classic Pig dice game built with HTML, CSS, and vanilla JavaScript.

## Overview

Pig Game is a two-player game where players take turns rolling a die to accumulate points. Players can choose to hold their current score and add it to their total, but rolling a 1 resets the current score and passes the turn to the other player. The first player to reach the winning score wins the game.

## Features

* Two-player gameplay
* Random dice roll generation
* Current and total score tracking
* Hold score functionality
* Automatic turn switching
* Winner detection
* New Game option to restart the game

## Programing languages Used

* HTML5
* CSS3
* JavaScript (ES6)
